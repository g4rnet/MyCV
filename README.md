## 🇬🇧 English

### Overview

A single-page, **print-ready CV** built entirely with semantic HTML5 and vanilla CSS3 — no frameworks, no JavaScript, no bloat. Styled with the iconic **Dracula dark theme** and **JetBrains Mono** typeface to give it a clean, developer-native aesthetic.

> Designed to look sharp both on screen and on paper (A4 format).

---

### ✦ What's inside

| File | Purpose |
|------|---------|
| `index.html` | Semantic CV structure using `<article>`, `<section>`, `<header>` |
| `style.css` | Dracula-themed stylesheet, A4 layout, CSS Grid & Flexbox |
| `icon-document.png` | Browser tab favicon |

---

### ✦ Tech & Design choices

- **Pure HTML + CSS** — zero dependencies, zero build tools
- **Dracula Color Palette** — `#FF79C6` pink, `#BD93F9` purple, `#50FA7B` green accents
- **JetBrains Mono** — imported via Google Fonts for a monospace "code" feel
- **A4 Layout** — `210mm × 297mm` fixed-width sheet, centered on page
- **CSS Grid** — used for the skills section (3-column layout)
- **Flexbox** — used for contact bar, language list, and section headers
- **Semantic HTML** — `aria-label` attributes on sections for accessibility
- **Print-friendly** — layout is structured for clean PDF export via browser print

### ✦ Quick Start

```bash
# Clone the repo
git clone https://github.com/your-username/cv-makarenko.git
```

To **export as PDF**: open in any browser → `Ctrl+P` / `Cmd+P` → Save as PDF.

## 🇺🇦 Українська

### Огляд

Односторінкове **резюме у форматі A4**, побудоване виключно на семантичному HTML5 та чистому CSS3 — без фреймворків, без JavaScript, без зайвого. Стилізоване в культовій темі **Dracula** та шрифтом **JetBrains Mono** для чіткого, девелоперського вигляду.

> Виглядає гарно як на екрані, так і при друці або експорті в PDF.

---

### ✦ Що всередині

|------|------------|

### ✦ Технічні та дизайн-рішення

- **Чистий HTML + CSS** — нульова залежність, нульові інструменти збірки
- **Кольорова палітра Dracula** — акценти `#FF79C6` рожевий, `#BD93F9` фіолетовий, `#50FA7B` зелений
- **JetBrains Mono** — підключено через Google Fonts для моноширинного "кодового" відчуття
- **A4 розмітка** — фіксована ширина `210mm × 297mm`, відцентрована на сторінці
- **CSS Grid** — для секції навичок (3 колонки)
- **Flexbox** — для контактного рядка, списку мов та заголовків секцій
- **Семантичний HTML** — атрибути `aria-label` на секціях для доступності
- **Підготовлено до друку** — структура розрахована на чистий PDF-експорт через браузер
